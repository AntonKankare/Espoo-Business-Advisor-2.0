Espoo Business Advisor

Espoo Business Advisor is a minimal, multilingual web application designed to help people in Espoo prepare for a business advisory meeting. The app guides the user through a short question-and-answer flow and generates a clear, structured summary that can be provided to an advisor.

Achievement

This project earned 2nd place in one of the main challenges at the Junction hackathon, demonstrating its practical value, clarity, and impact.

Features

Multilingual interface

Simple and intuitive Q&A flow

Automatically generated, well-structured summary for business advisors
